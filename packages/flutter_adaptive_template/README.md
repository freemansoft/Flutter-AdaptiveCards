# Flutter Adaptive Template

A template engine for adaptive cards, enabling data binding and dynamic rendering of adaptive card payloads in Flutter.

## Status: Not yet implemented

- `$root` support for it being the root of the the data tree.  Only support separate data and template

For more information, please visit the [Flutter-AdaptiveCards GitHub Repository](https://github.com/freemansoft/Flutter-AdaptiveCards).
